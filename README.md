# ActorProject
- 연극 '너 혹시 그런 생각 해본 적 있어?'의 관객 경험 향상을 위해 제작한 프로그램으로, 실시간 모션 캡쳐를 사용한다.

# 프로젝트 정보
- 플랫폼 : Windows PC
- 엔진 : Unreal Engine 5
- 언어 : C++

# 기능
  - 'Movella'의 Xsesn Dot과 Xsens Dot SDK를 활용한 실시간 모션 캡처 기능
  - 연극에서 사용하기 위한 씬 전환 기능
